# OpenGL

* Download the glfw library files from: https://www.glfw.org/
* Copy the correct library folder into Dependencies\\GLFW
* If needed, update the additional library directories of the project in Properties -> Linker -> General
* Download the GLEW library files from: https://glew.sourceforge.net/
* Copy the correct library folder into Dependencies\\GLEW
* If needed, update the additional library directories of the project in Properties -> Linker -> General
